
# Portfolio — [ Nicky FLores]

Sitio web profesional con galería interactiva, venta de obras
y solicitud de diseños personalizados (logos y tatuajes).

---

## Stack


- HTML5 semántico
- CSS3 con variables y animaciones
- JavaScript vanilla
- GSAP para animaciones avanzadas

---

## Estructura del proyecto
portfolio-arte/
├── index.html
├── css/
│   ├── styles.css       → estilos base y variables
│   └── animations.css   → animaciones
├── js/
│   ├── main.js          → lógica general
│   └── galeria.js       → lógica de la galería
├── img/
│   └── obras/           → imágenes del artista
├── data/
│   └── obras.json       → catálogo de obras
└── assets/
└── logo.svg

---

## Cómo agregar una obra nueva

1. Copia la imagen a `/img/obras/`
2. Abre `data/obras.json`
3. Agrega un bloque nuevo siguiendo la estructura existente
4. Guarda y sube los cambios a GitHub

---

## Categorías válidas

`tatuaje` · `logo` · `ilustracion` · `otro`

---

## Ver el proyecto localmente

Abre `index.html` en el navegador o usa Live Server en VS Code.
